# AI Trading Platform

A modern, professional trading platform built with React, TypeScript, and Tailwind CSS.

## Features

- **Dashboard**: Overview of account balance, recent orders, and market data
- **Trading Interface**: Advanced trading with real-time market data
- **Portfolio Management**: Create and manage multiple portfolios
- **Market Data**: Real-time stock prices and market information
- **Order Management**: Track and manage trading orders
- **Account Management**: View account details and transaction history
- **Research Tools**: Advanced charting and analysis tools

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS with shadcn/ui components
- **Charts**: Recharts for data visualization
- **Icons**: Lucide React
- **Notifications**: Sonner for toast notifications

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd my-figma-project
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

### Available Scripts

- `npm start` - Starts the development server
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite
- `npm eject` - Ejects from Create React App (not recommended)
